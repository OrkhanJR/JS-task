TASK 2

The courier service delivers at a single rate: 5 AZN fixed + 0.25 AZN for each kilometer of the road. Write a function that will prompt the user for the distance in km and return the cost of delivery. 

            
let price = 5;
let costPerKm = 0.25;
let distance = prompt('please write a distance in km');

const deliveryCost = price + costPerKm * distance;

alert(`The cost of delivery is ${deliveryCost} AZN.`)
