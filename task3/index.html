TASK 3

Each product in the online store receipt is described by the following values: - Name; - Unit price; - Quantity; - Participates in the promotion (yes/no). Write a function that receives the listed values and calculates the receiptâ€™s amount according to the following rules: - If the quantity of the product is greater than or equal to 5, then a 10% discount is applied to the product. - If the product participates in the promotion, a 15% discount is applied. Discounts may be cumulative. 

            
              function calculateReceiptAmount(name, unitPrice, quantity, participatesInPromotion) {
                let discount = 0;
                let totalAmount = unitPrice * quantity;
              
                if (quantity >= 5) {
                  discount += 0.1;
                }
              
                if (participatesInPromotion) {
                  discount += 0.15;
                }
              
                const discountedAmount = totalAmount - (totalAmount * discount);
                
                return discountedAmount;
              }
